#  Mini Dictionnaire

le projet que j'ai nomme mini- dictionnaire web est un projet que j'ai developpe en utilisant les languages **HTML, CSS, JS** 
**HTML** a pour role de de structurer la page. 
**CSS** a pour role de styliser ma page web
**JS** Strategie back-end (ogique du code et bien d'autres astuces pour le deevloppement de mon mini-dictionnaire)

# Fonctionnalite 
- le projet mini-dictionnaire a pour role principale d'aider des personnes d'origine de l'ouest Cameroun a traduire les mots de politesse, de la tech en leur langue vernaculaire (GHOMALA);

# structure du projet 

- le projet comporte 4 fichiers, soit:
* un HTML
* un CSS
* un Readme 
* un JSON
